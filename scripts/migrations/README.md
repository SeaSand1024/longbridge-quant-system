# 数据库迁移脚本

此目录包含历史数据库迁移脚本，用于记录数据库结构的演进历史。

## 迁移脚本列表

| 脚本 | 功能 | 执行时间 |
|------|------|----------|
| `add_test_mode_fields.py` | 添加 test_mode 字段到相关表 | 历史迁移 |
| `migrate_add_group.py` | 添加 group 字段 | 历史迁移 |
| `migrate_add_type.py` | 添加 type 字段 | 历史迁移 |

## 注意事项

⚠️ **这些是历史迁移脚本，仅用于记录数据库演进历史。**

- 如果需要初始化新数据库，请使用 `../init_db.py`
- 不要重复执行这些历史迁移脚本
- 新的数据库变更应创建新的迁移脚本并记录在此目录
